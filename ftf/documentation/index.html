Travpro - Responsive HTML5 Template

  • created: 01/20/2015
  • latest update: 01/28/2015
  • by: 1stone

Thank you for purchasing Travpro Landing page Template.
“Travpro” is a Tourism Landing page Template which have clean & modern design with a Call To Action button on top banner.

HTML

This template, which is a responsive layout for tablet and other mobile devices, is based on the HTML5 structure and Twitter Bootstrap 3. Visit http://getbootstrap.com/ to learn more about Bookstrap. For a detailed information about its grid system, click on http://getbootstrap.com/css/#grid.

As a brief explanation, the grid is composed of 12 columns; each named as "col-xs-1", "col-xs-2", "col-xs-3", "col-xs-4", and so on and so forth. The column divisions are grouped in rows.

The grid system works for multiple devices. Please check the table below.

  Extra Small Devices(<481) Intermediate Small Devices(≥481 and < 768) Small Devices(≥768) Medium Devices(≥992) Large Devices(≥1200)
Container width Auto  
Class Prefix col-xs- col-sms- col-sm- col-md- col-lg-

CSS

All css files are located in the /css.
 

- bootstrap/css
  • bootstrap.css - Generic bootstrap styles
- css
  • style.css - Main style file (all styles are located in this file)
  • custom.css - Custom style file. We would recommend you to write your rules and style declarations here.
  • updates.css - Updated styles for further versions
  • animations.css - css3 animation styles
  • loader.css - preloader style

Javascript

This template imports several JS files.

The Javascript  files are loaded both in the header and footer. Several files are loaded in the header but most of them are loaded at the bottom to increase the site speed and rendering.

The Javascript files used in this template are the following:

  1. jQuery library
  2. Modernizr JS
  3. Bootstrap JS  (bootstrap/js/bootstrap.min.js)
  4. Waypoints JS
  5. Mousewheel JS
  6. Smoothscroll JS
  7. Travpro main scripts (custom.js) - It contains all the scripts we are using in the template. To get a detailed understanding of this, please have a look at this file that contains an explanation written in comment.

Customization

Changing Logo

Change a default logo by changing logo.png in the /images directory, or by opening the page and changing the following code.

<div class="header-top-outer clearfix">
	<div class="logo">
		<a href="#">
			<img src="images/logo-1.png" class="img-responsive" alt="TarvPro HTML5 Template"/>
		</a>
	</div>
</div>

Change the logo in /images/logo.png.

Changing Banner

Change a default banner by changing header-bg-01.jpg in the /images directory, or by opening the page and changing the following code.

<header>
	<div class="banner-outer">
		<img src="images/header-bg-01.jpg" class="banner-img" alt="TarvPro HTML5 Template" />
	</div>	
</header>

Change the banner in /images/header-bg-01.jpg.

Changing What We Do Content

Change default content by changing the following code.

		<div class="row">
			<div class="col-sm-4 col-xs-12">
				<div class="what-inner-wrapper">
					<div class="what-img-wrapper">
						<!-- PROCEDURE IMAGE GOES HERE -->
					</div>
					<h3><!-- PROCEDURE HEADING GOES HERE --></h3>
					<p>
						<!-- PROCEDURE TEXT GOES HERE --> 
					</p>
				</div> 
			</div>	
		</div>
	</div>
</section>S

Change the procedure images in /images/yellow-what-img-01.png.
